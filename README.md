# Tasklist

To run simply enter in terminal:
`npm i && npm run dev`

Main stack:

- React, Typescript, Tailwind, Shad-cn, React-Query, Zustand
- Under the hood Vite bundler. It uses [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) with [SWC](https://swc.rs/) for Fast Refresh
